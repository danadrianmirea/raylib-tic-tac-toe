# Raylib Tic Tac Toe

## Description

This is a simple Tic Tac Toe game made with [Raylib](https://www.raylib.com/).

## Building

```sh
mkdir cmake-build
cd cmake-build
cmake ..
make
./raylib_tic_tac_toe
```

## Development

Install [Raylib](https://www.raylib.com/) with [Homebrew](https://brew.sh/).

```
brew install raylib
```
